# quant-research
A collection of Python tools for quantitative trading research and risk analysis.  

This repository explores key concepts in systematic trading such as Monte Carlo simulations, risk of ruin modeling, drawdown analysis, and position sizing experiments.

Projects

1. Monte Carlo Risk Simulator
   Simulates trading strategy outcomes to analyze variance, drawdowns, and risk of ruin.

2. EMA Strategy Backtester
   A simple backtesting engine that evaluates an EMA crossover strategy using historical market data.
